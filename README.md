# Robot Fleet Monitor

A Python-based simulation of a robotic fleet management system used in manufacturing environments.

## Features
- Simulates multiple robots and their states
- Tracks battery levels and operational status
- Generates alerts for low battery conditions
- Demonstrates basic fleet monitoring concepts

## Why I Built This
This project was created to explore robotics software concepts such as fleet management, telemetry, and monitoring systems in a production environment.

## How to Run
```bash
python main.py
```

## Example Output

Robot_2 | LOW BATTERY | Battery: 15%
